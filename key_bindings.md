# Key bindings

```
<leader> : ,

f  : find
t  : (find)to
ct : change+to+char
cf : change+find+char

H  : Go top
M  : Go middle
L  : Go bottom

iw : in word
ip : in paragraph
i" : in ""
it : in tag
as :

cp : system copy
cP : system copy line
cv : sytem paste

gu : make lowercase
gU : make uppercase

vsp : split vertical
sp  : split horisontal

<C-h> : split nav (left)
<C-j> : split nav (down)
<C-k> : split nav (up)
<C-l> : split nav (right)

<C-d> : down half page
<C-u> : up half page

<C-f> : down a page
<C-b> : up a page

<C-n> : multiple cursors

ds : remove surroundings
csiw : change surrindings in word
yssb : warp line with ()

gs2j : sort two lines down
gsip : sort current paragraph
gsii : sort current indentation

<leader>d : NERDTree
<leader>f : Find current in NERDTree
<leader>space : strip whitespace
<leader>i : indent file

<Tab> : next buffer
<S-Tab> : prev buffer

<leader>; : search for file
; : search for git file
\ : search in files

<leader>s : Go to def (ale)
<leader>a : Find refs (ale)
```
